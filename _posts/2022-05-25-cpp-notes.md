### `istringstream` from `sstream`:
This helps to get word from a string separated by a space.

```cpp
...
std::string s = "foo bar";
std::istringstream p_s(s);

while(p_s) {
	p_s >> word; # get the word to the variable
	std::cout<<word<<"\n";
}
...
## Output ##
foo
bar
```
